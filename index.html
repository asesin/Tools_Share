UE5 Material Editor — 사용 가이드 ✕
📐 기본 조작
1
노드 추가 — 우클릭 컨텍스트 메뉴 또는 왼쪽 Palette 탭에서 클릭
2
노드 이동 — 노드 클릭 후 드래그. 선택 후 다른 노드도 드래그 가능
3
다중 선택 — Ctrl+클릭 또는 빈 캔버스에서 드래그해서 박스 셀렉션
4
핀 연결 — 핀을 드래그해서 호환되는 핀에 연결. 드래그 중 연결 가능한 핀이 하이라이트됨
5
와이어 삭제 — 연결된 와이어 클릭
6
노드 속성 — 노드 선택 후 Details 탭에서 파라미터 편집
🗺 뷰포트 조작
↕
줌 — 마우스 휠 (커서 위치 기준)
✋
패닝 — Alt+드래그 또는 중간 마우스 버튼
F
전체 보기 — F키 또는 툴바 ⊞ Fit 버튼
⌨ 단축키
Ctrl+S 저장 (.json) Ctrl+O 불러오기 (.json) Ctrl+N 새 머터리얼 Ctrl+A 전체 선택 Del 선택 노드 삭제 F 전체 Fit H 노드 레퍼런스 도움말 V 그래프 검증 C 코멘트 박스 추가 Home 뷰 초기화
💾 저장 / 불러오기
1
저장 — Ctrl+S 또는 툴바 💾 버튼 → .json 파일 다운로드
2
불러오기 — Ctrl+O 또는 툴바 📂 버튼 → 저장된 .json 선택
저장 파일에는 노드 위치, 타입, 파라미터, 와이어 연결, 코멘트, 뷰 위치가 모두 포함됩니다.
🐍 UE5 Python Export
1
툴바 ↗ Python 버튼 클릭 → 스크립트 생성
2
Unreal Editor → Tools → Execute Python Script → 텍스트 박스에 붙여넣기
3
실행하면 /Game/Materials/ 에 머터리얼 에셋이 자동 생성됩니다
⚠ HTTPS 환경(로컬 파일 실행 시)에서는 클립보드 복사가 차단될 수 있습니다. 이 경우 팝업 텍스트 박스에서 직접 복사하세요.
⚠ 텍스처 에셋(/Game/Textures/)은 별도로 언리얼에 임포트 후 MaterialInstance에서 지정해야 합니다.
📦 프리셋
1
Presets 탭에서 원하는 프리셋을 클릭해 선택 (다중 선택 가능)
2
Apply Combo 버튼 → 선택한 프리셋들이 자동 병합되어 캔버스에 배치
블렌드 모드 충돌 시 Additive > Translucent > Masked > Opaque 순서로 우선순위가 적용됩니다.
